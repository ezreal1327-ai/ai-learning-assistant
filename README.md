# AI Learning Assistant

An AI-powered assistant designed to enhance students’ learning efficiency through intelligent support in writing, language learning, and study planning.

## Features
- AI writing assistance  
- English learning support  
- Study planning suggestions  

## Tech Stack
- React  
- JavaScript  
- OpenAI API  

## Description
This project explores how AI can support students in writing, language learning, and productivity.
