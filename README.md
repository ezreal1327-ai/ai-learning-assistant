# AI Learning Assistant

An AI-powered learning assistant designed to improve students' productivity in writing, language learning, and study planning.

## 🚀 Features
- AI writing assistance for essays and assignments  
- English learning support (grammar, rewriting, vocabulary)  
- Study planning and productivity suggestions  
- Chat-based interaction interface  

## 🛠 Tech Stack
- React  
- JavaScript  
- OpenAI API  

## 📌 Project Description
This project explores how AI can support students in academic writing, language learning, and daily study tasks. It aims to provide simple and practical AI tools to improve learning efficiency.
